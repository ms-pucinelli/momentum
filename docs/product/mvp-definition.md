# MVP Features

## Habits

* Create habits
* Edit habits
* Archive habits
* Mark habits as completed

## Gamification

* Earn points
* Track weekly progress
* Configure weekly goals
* Support repeatable habits (example: drinking water)

## Emotional Journal

* Rate the day from 0 to 10
* Write an optional reflection
* View previous journal entries

## Navigation

* Dashboard
* My Streak
* My Journal

---

# Technical Constraints

For the MVP:

* No backend
* No user authentication
* No cloud synchronization
* Data stored locally using Local Storage
* Built with HTML, CSS and JavaScript only
